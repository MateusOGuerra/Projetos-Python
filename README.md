# Sistema de Suporte ao Cliente

Projeto desenvolvido em Python por Mateus Guerra.

## Descrição

Este sistema simula um atendimento ao cliente através de um menu interativo no terminal.

## Funcionalidades

* Consulta de produtos
* Consulta de endereço
* Consulta de horário de funcionamento
* Atendimento ao cliente
* Registro de clientes em arquivo de texto
* Tratamento de opções inválidas

## Tecnologias Utilizadas

* Python 3
* VS Code

## Como Executar

1. Abra o projeto no VS Code.
2. Abra o terminal.
3. Execute o comando:

python main.py
